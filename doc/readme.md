# Espace collaboratif mobile

Le projet utilise le framework open-source [Cordova](https://cordova.apache.org/docs/en/latest/guide/platforms/index.html) développé par la Fondation Apache.    
Celui-ci permet de créer des applications pour différentes plateformes (Android, iOS, Windows mobile) en HTML, CSS et JavaScript.

## Application CordovApp

Afin de simplifier l'utilisation de Cordova et d'unifier les développement, l'application utilise la base applicative [CordovApp]{@link module:CordovApp}.

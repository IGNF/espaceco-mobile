Module: Ripart

Fichier : ./js/ripart.js

Accès au service de remontées d'information de l'espace collaboratif

La classe RIPart permet d'accéder au service de remontées d'information de l'espace collaboratif.
Elle utilise l'API de l'espace collaboratif.

Organisation

  • Le fichier .js/ripart.js définit la classe de base pour l'utilisation du service.
  • La classe est étendue dans le fichier .js/wapp.ripart.js qui utilise les fonctinnalités de la CordovApp pour la gestion des remontées (dialogue de connexion, formulaires, etc.).
    Elle gère également un tableau de remontée local.
